## Getting Started

To clone the repository use
```bash
git clone https://github.com/chirag-ganguli/seed-store
```

To run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results.
